# WiserCare Playwright Automation

Exploratory test automation suite for WiserCare patient decision modules with automatic bug report generation.

## What It Tests
- Registration form validation
- Advance Care Planning module
- Prostate Cancer Spanish module
- Mobile responsiveness
- Broken images detection
- Console error detection
- Broken link detection

## Results
- 20/20 tests passing
- 0 bugs found
- Auto-generated bug reports in markdown format

## Tech Stack
- Playwright
- TypeScript
- GitHub Actions

## Run Tests
npm run test:registration
npm run test:modules
npm run test:all
